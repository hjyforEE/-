## TetrisInMomochromic_基於陣列邏輯之單色版俄羅斯方塊
#### 使用pygame作為遊戲引擎，在replit雲端平台進行編寫及執行測試。
#### 使用本地版llama3_7b、copilot協助進行編譯及除錯
#### 可使用micropython之形式運行於meshtasic(ESP32+oled)、CRTviewfinder等可攜式單色顯示裝置，不受多彩顯示之硬體限制。
